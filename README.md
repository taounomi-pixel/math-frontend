# MathVis - 定理可视化平台 (Frontend)

MathVis 是一个致力于用代码和动画解析硬核数学定理的可视化社区。本项目为平台的前端部分，为用户提供沉浸式的数学推导与视频观看体验。


## 技术栈
* **核心框架：** React
* **构建工具：** Vite (极速冷启动与热重载)
* **样式引擎：** Tailwind CSS (原子化构建优雅 UI)
* **线上部署：** Vercel (Serverless 边缘网络托管)

## 本地运行指南
1.  克隆仓库并进入目录：
    ```bash
    git clone [https://github.com/taounomi-pixel/math-frontend.git](https://github.com/taounomi-pixel/math-frontend.git)
    cd math-frontend
    ```
2.  安装纯净依赖包：
    ```bash
    npm install
    ```
3.  启动本地开发服务器：
    ```bash
    npm run dev
    ```
